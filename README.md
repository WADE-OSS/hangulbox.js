# kop.js

한글을 다양한 형태로 바꿀 수 있는 자바스크입트 라이브러리 입니다.

<br>
<br>
<br>

### 한단어만 분리하기
```js
  kop.wordSeparate('안');
  // return 'ㅇㅏㄴ'
```

### 문장을 분리하기
```js
  kop.wordSeparateAll('안녕하세요');
   // return 'ㅇㅏㄴㄴㅕㅇㅎㅏㅅㅔㅇㅛ'
```

### 문자을 분리하고 Array로 받기
```js
kop.wordSeparateToArray('안녕하세요');
/* return [
0: (3) ['ㅇ', 'ㅏ', 'ㄴ']
1: (3) ['ㄴ', 'ㅕ', 'ㅇ']
2: (3) ['ㅎ', 'ㅏ', '']
3: (3) ['ㅅ', 'ㅔ', '']
4: (3) ['ㅇ', 'ㅛ', '']
] */
```
